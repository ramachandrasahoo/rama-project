Frontend Devlopment by rama

Frontend Devlopment is the devlopment of the graphical user interface of a website, through the use of HTML, CSS, and Javascript, so that users can view and interact with that website.

  • HTML: The HyperText Markup Language or HTMLis the standard markup language for the documents design to be displayed in a web broweser.
  • CSS: Cascadin Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML.
    1. Pure CSS
    2. Bootstrap (Framework)
    3. Tailwind (Framework)
    4. Saas (Preprocessor)
  • Javascript: JavaScript often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is use to add functionality in the website.
    1. Vanilla JS
    2. jQuery (Library)
    3. React (Framework)
    4. Angular (Framework)
    5. Vue (Framework)
    6. Typescript (Preprocessor)
